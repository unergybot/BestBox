# BestBox Product Overview

BestBox is an enterprise-grade agentic applications demonstration kit that showcases how AI agents can transform traditional ERP/CRM/OA workflows. The system demonstrates production-ready multi-agent architecture patterns using locally-deployed LLMs.

## Core Value Proposition

- **100% Local Deployment**: All models and data stay on-premise for security and compliance
- **AMD ROCm Native**: Optimized for AMD GPUs without CUDA dependency
- **Enterprise Integration**: Direct integration with ERP, CRM, IT Operations, and Office Automation systems
- **Non-Technical User Focus**: Natural language interface accessible to business users

## Key Features

### Multi-Agent System
- **Router Agent**: Intelligent query classification and routing
- **ERP Agent**: Finance, procurement, inventory, invoices, vendor management
- **CRM Agent**: Sales, leads, customers, deals, opportunities, revenue tracking
- **IT Ops Agent**: Server monitoring, error logs, alerts, maintenance
- **OA Agent**: Email, scheduling, meetings, calendar, documents, leave requests
- **General Agent**: Greetings, help requests, cross-domain queries

### Technical Capabilities
- Real-time streaming responses via CopilotKit protocol
- Voice input/output support with LiveKit integration
- Document and image upload processing
- Generative UI components for dynamic interfaces
- Context management with sliding window algorithm
- Comprehensive observability and monitoring

## Target Users

- **Business Users**: Non-technical staff needing AI assistance for daily workflows
- **IT Administrators**: System operators managing enterprise infrastructure
- **Developers**: Teams building AI-powered enterprise applications
- **Decision Makers**: Executives evaluating AI integration strategies

## Demo Scenarios

1. **ERP Copilot**: Invoice processing, inventory checks, financial reporting
2. **CRM Sales Assistant**: Lead qualification, quotation generation, opportunity tracking
3. **IT Operations Agent**: Ticket routing, knowledge base search, automated diagnostics
4. **OA Workflow Agent**: Leave approvals, meeting scheduling, document workflows
